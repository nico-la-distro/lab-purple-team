## Objectif

Organiser les objets Active Directory de manière logique pour faciliter l’administration et l’application des GPO.
## Arborescence

Chaque objet Active Directory appartient à une seule OU.

```
LAB
├── USERS
│   ├── alice
│
├── WORKSTATIONS
│   ├── WS01
│
├── SERVERS
│   ├── DC01
|
|-- SERVICES
|   |-- svc-web
|   |-- svc-backup
|   |-- svc-sync
```
