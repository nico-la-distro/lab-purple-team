## Installation

1. Install Windows Server 2022 Standard Evaluation (Desktop Experience)
2. Configure static IP & DNS (Host-Only)
3. Install AD DS
4. Create domain: **lab.local**
5. Promote as Domain Controller
6. Change name: DC01
7. ping 192.168.10.100 & 8.8.8.8 ✅
8. nslookup lab.local -> Address:  192.168.10.100 ✅
